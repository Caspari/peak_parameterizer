Class ResultsContainer

source code

A data container with a three dimensional matrix.

X: Window sizes Y: Slope thresholds Z: Error values

The matrix is indexed using the following scheme: window[window_size][slope_threshold][error_value]

Parallel lists (window_sizes, slope_thresholds, error_values) serve as axes.
